# 飞鼠AI招聘助手 - 产品功能文档

## 产品概述

飞鼠AI招聘助手是一款智能AI-ATS系统，通过Agent招聘工作流实现前链路寻访与初筛工作的人力替代，帮助企业提升招聘效率，降低人力成本。

## 核心价值主张

- **智能提效**：AI自动化处理简历筛选、候选人寻访等重复性工作，招聘效率提升300%
- **精准匹配**：基于岗位画像与候选人画像的深度匹配，入职转化率提升45%
- **全流程覆盖**：从JD生成到面试评分，全链路AI赋能，打造闭环招聘体验

## 主要功能模块

### 1. 智能岗位JD生成
- AI基于岗位需求自动生成结构化JD
- 支持多行业、多职位类型的JD模板
- 一键优化现有JD，提升职位吸引力

### 2. 多平台寻访候选人
- 自动同步BOSS直聘、猎聘、智联招聘等主流招聘平台
- AI智能筛选目标候选人池
- 支持自定义寻访策略，定向触达

### 3. 自动下载解析简历入库
- 一键下载候选人简历
- AI解析简历关键信息，结构化存储
- 自动去重，避免重复入库

### 4. AI简历匹配度打分
- 基于岗位画像与候选人画像的深度匹配
- 多维度评估：教育背景、工作经历、技能匹配度
- 自动生成评分报告，辅助HR决策

### 5. 高分候选人自动邀约
- 阈值自动触发邀约流程
- 支持自定义邀约话术模板
- 多渠道触达：短信、邮件、站内信

### 6. AI生成面试题库
- 根据岗位要求智能生成面试题
- 提供高中低分回答范例
- 支持结构化/非结构化面试题库

### 7. 面试录音AI评分
- 自动录音并转写面试内容
- AI分析候选人回答质量
- 多维度评分：专业能力、沟通表达、逻辑思维

### 8. AI决策推进环节
- 自动推进招聘流程节点
- 智能提醒面试安排、Offer跟进
- 数据看板实时追踪招聘进度
